The For In Loop The JavaScript for in statement loops through the properties of an Object: Syntax for (key in object) { // code block to be executed } Example const person = {fname:"John", lname:"Doe", age:25}; let text = ""; for (let x in person) { text += person[x]; } JavaScript Loops Loops are handy, if you want to run the same code over and over again, each time with a different value.
